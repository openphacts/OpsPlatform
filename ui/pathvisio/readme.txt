To run this demo:

- make sure you have larkc running
- launch the datasetsworkflow.ttl
- this should be launched as the first workflow so that you're running it on http://localhost:8183
- run pathvisio in this directory with the following command line

java -jar OpsPathvisio.jar -p org.pathvisio.plugins.PppPlugin

- once running you can load the ops.gpml file in this directory (file open)
- by right clicking on the node 7410... and going to Pathway Loom and then OpenPhacts you can retrieve some suggestions for that compound
